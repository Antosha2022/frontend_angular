  1. Determine the width and height in millimeters for the individual card size for bulk printing.
  2. Choose a version. By default, the system chooses the latest version, and you can easily switch to the one you need.
  3. Select a sequence of tags (or add your own) in the order in which you want them printed.
  4. The template must contain at least 1 tag.
  5. All tags will be added to the template with a space.
  6. Add teg '@' if you need to start on a new line
  7. Press Print
  8. PDF file for print will open in new tab
